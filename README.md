# DJANGObabies
```bash
python3 -m venv venv #Crear entorno virtual

$ source venv/bin/activate # Iniciar entorno virtual

$ pip3 install django #Instalar django

$ django-admin startproject babies #Crear proyecto 'babies'

$ python3 manage.py startapp babyevents #Crear la app babyevents

$ python3 manage.py migrate #Migrar cambios a modelos

$ python3 manage.py makemigrations 

```
